# PWA Icons

Generate the following icons for full PWA support:

1. **favicon.ico** (32x32) — Browser tab icon
2. **icon-192.png** (192x192) — Android home screen
3. **icon-512.png** (512x512) — Android splash screen
4. **apple-touch-icon.png** (180x180) — iOS home screen

## Design guideline
- Background: Dark blue (#0b1220)
- Foreground: White/cyan gradient "WR" letters
- Style: Hexagon or rounded square, gaming aesthetic
- Tools: Use [Favicon.io](https://favicon.io) or [RealFaviconGenerator](https://realfavicongenerator.net)

## Quick generation
1. Go to https://favicon.io/favicon-generator/
2. Text: "WR"
3. Background: #0b1220
4. Font: Bold, sans-serif
5. Download and place files in `/public/`
